# Running the Data Science Encyclopedia

## Quick Start

1. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Flask server:**
   ```bash
   python app.py
   ```

3. **Open your browser:**
   Navigate to: `http://localhost:5002`
